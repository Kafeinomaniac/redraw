# Contributing

## Setup

### Install dependencies (OS X)

```Shell
npm i
```
### The following can help to avoid error messages, during installation
```Shell
brew install pkg-config cairo libpng jpeg giflib
```

### Run tests
```Shell
npm run test
npm run tdd
npm run eslint
```

### Development mode
```Shell
npm run dev
```

### Build
```Shell
npm run build
```
