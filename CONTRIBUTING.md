# Contributing

## Setup

### Install dependencies (OS X)

```Shell
brew install pkg-config cairo libpng jpeg giflib
```

### Run tests

```Shell
npm test
```

### Watch
```Shell
gulp watch (or)

webpack-dev-server
```

### Build
```Shell
gulp build (or)

webpack --progress --colors
```
