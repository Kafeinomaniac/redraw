# redraw JS
Simple HTML5 Chart tools for the awesome [FabricJs](http://fabricjs.com/).

## Setup
### For development
```Shell
npm test
gulp watch

gulp build
```

###
```Shell
brew install pkg-config cairo libpng jpeg giflib
```
### Build
```Shell
webpack --progress --colors
```
